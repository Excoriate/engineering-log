# Shared kubeconfig is a moving pointer

DEV and ACC tabs can display different labels while still sharing one mutable kubeconfig. A context switch in either tab can redirect later commands in the other.

Reusable rule: bind a context once, prove its API, pass `--context` on every query, and re-check the API before every evidence block. If identity changes or a guard fails, discard the whole block rather than keeping convenient partial output.

The AVD typing path corrupted complex punctuation during automation. That does not weaken the control; it limits the proof tier. The wrapper is structurally verified, but it must be human-pasted or run with an isolated kubeconfig before Wednesday's status can become monitor-ready.
